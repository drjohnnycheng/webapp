Text Analytics Platform 文本分析平台 (Version 0.7.0)
Help Page 協助指引 (Updated 更新日期 : 17 March 2025)

Main Menu 選單

1. Scripture Browsing with Exegesis 經文瀏覽 (附註解)
   * This function provides the prescribed scripture for the user's browsing and display, if needed,the relevant book's or verses's exegesis.
   * 這功能讓讀者瀏覽指定經文並按需要查閱有關書卷或經文的註解。

2. Fuzzy Search 近似字串搜尋
   * This function searches for similar terms/phrases on the selected text.
   * 這功能在選取的文本中搜尋近似的指定語句。

3. Scripture Statistics 經文統計數據
   * This function shows aggregated figures on the numbers of chapters and verses of Bible books and categories.
   * 這功能顯示聖經書卷及書卷類別的章/節匯總數據。

4. Quotes from Old Testament (OT) 引用舊約經文
    * This function attempts to identify the relevant quoted OT scripture based on the prescribed NT verses.
    * 這功能根據個別新約經文，嘗試尋找相關引用的舊約經文。

5. Wordcloud 文字雲
   * This function converts a prescribed scripture into a word cloud. (A word cloud is a visual representation (image) of word data.)
   * 這功能將指定經文轉換為文字雲。(文字雲是關鍵詞的視覺化描述。)

6. Text Summary 文本摘要
   * This function extracts an important subset of the prescribed sentences / Bible scripture as a summary.
   * 這功能從指定文章或經文抽取部份較為重要的句子作為摘要。

7. Topic Modeling (LDA, NMF, BERTopic) 主題建模
   * This function builds the prescribed topic model of the prescribed documents / Bible scripture.
   * 這功能根據指定的文本/經文去建立指定的主題模型。

8. Show Module Code (Python) 查閱模組程式碼
   * This function shows the source code (program code) of a prescribed module function or all the functions of a prescribed module.
   * 這功能顯示指定模組功能或指定模組所有功能的原始碼。
